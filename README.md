# Luan Nicolau — Portfolio

**Live:** [https://ln-nico-portfolio.vercel.app](https://ln-nico-portfolio.vercel.app)

---

## English

Personal portfolio website showcasing projects, skills, and contact information. Built with vanilla HTML, CSS, and JavaScript—no frameworks—with a focus on performance, accessibility, and responsive design.

### Features

- **Multi-language (i18n):** PT-BR, PT-PT, and EN. Language choice is stored in `localStorage` and applied on load.
- **Sections:** Hero, About, Projects (with links to live demos and source code), and Contact.
- **Responsive:** Layout adapts to mobile, tablet, and desktop; includes a mobile hamburger menu.
- **Design:** Custom typography (Fraunces, Outfit), subtle noise overlay, and smooth scroll/animations.

### Tech stack

- HTML5
- CSS3 (custom properties, Grid, Flexbox)
- Vanilla JavaScript (no build step)

### Running locally

1. Clone the repo and open the project folder.
2. Serve the files with any static server (e.g. VS Code Live Server, or `npx serve .`).
3. Open the URL shown (e.g. `http://localhost:3000`) in your browser.

### Deployment

Configured for [Vercel](https://vercel.com) via `vercel.json`. Push to your connected repo or run `vercel` in the project root to deploy.

## Português (PT-BR)

Site de portfólio pessoal com projetos, habilidades e informações de contato. Desenvolvido com HTML, CSS e JavaScript puros—sem frameworks—com foco em performance, acessibilidade e design responsivo.

### Funcionalidades

- **Multi-idioma (i18n):** PT-BR, PT-PT e EN. A escolha do idioma é salva no `localStorage` e aplicada ao carregar.
- **Seções:** Hero, Sobre, Projetos (com links para demos e código-fonte) e Contato.
- **Responsivo:** Layout adaptado para mobile, tablet e desktop; inclui menu hambúrguer no mobile.
- **Design:** Tipografia customizada (Fraunces, Outfit), overlay de ruído sutil e rolagem/animações suaves.

### Stack

- HTML5
- CSS3 (variáveis, Grid, Flexbox)
- JavaScript puro (sem etapa de build)

### Rodar localmente

1. Clone o repositório e abra a pasta do projeto.
2. Sirva os arquivos com um servidor estático (ex.: Live Server no VS Code ou `npx serve .`).
3. Abra o endereço indicado (ex.: `http://localhost:3000`) no navegador.

### Deploy

Configurado para [Vercel](https://vercel.com) via `vercel.json`. Faça push para o repositório conectado ou execute `vercel` na raiz do projeto para publicar.
